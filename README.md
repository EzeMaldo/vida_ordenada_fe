## рҹ”„ Proxy Backend

`src/app/api/backend/[...path]/route.ts` es un catch-all que reenvГӯa todas las peticiones a `NEXT_PUBLIC_API_URL`. Esto permite:

- вң… Sin problemas de CORS
- вң… URL del backend fuera del bundle del cliente
- вң… Posibilidad de agregar middleware server-side (rate limiting, logging)

```ts
// El cliente llama a:
fetch('/api/backend/auth/login', { ... })

// El proxy reenvГӯa a:
fetch('https://vidaordenadabe-production.up.railway.app/auth/login', { ... })
```
